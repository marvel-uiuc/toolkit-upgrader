Toolkit Upgrader

This tool converts HTML code with v2 Illinois web toolkit components into their equivalent v3 components. With regular HTML the results are usually correct right away, but when converting template languages (eg. Twig) that are not valid HTML you may need to double check the results. You can paste in fragments of HTML or full pages. If you run into problems or have requests, feel free to create an issue in GitHub.

Copy & paste your HTML to the first box and click convert.
